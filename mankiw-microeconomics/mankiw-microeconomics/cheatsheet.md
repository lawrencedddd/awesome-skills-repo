# Mankiw Microeconomics Cheatsheet

## Decision Rules

| Situation | Rule |
|---|---|
| Take an action? | Act if marginal benefit >= marginal cost. |
| Specialize production? | Assign to the lower opportunity-cost producer. |
| Market shock? | Demand/supply -> movement direction -> new equilibrium. |
| Who bears a tax? | The less elastic side bears more burden. |
| Competitive firm output? | Produce where P = MC. |
| Short-run shutdown? | Continue if P >= AVC. |
| Long-run exit? | Stay if P >= ATC. |
| Monopoly output and price? | Set quantity where MR = MC, then price from demand. |
| Labor hiring? | Hire until VMPL = W. |

## Market Failure Diagnosis

| Symptom | Likely mechanism | First question |
|---|---|---|
| Pollution, congestion, spillovers | Externality | Does price include third-party costs or benefits? |
| National defense, basic research | Public good | Is it nonexcludable and nonrival? |
| Fisheries, crowded roads | Common resource | Is it nonexcludable but rival? |
| Used cars, insurance, credit | Asymmetric information | Hidden action or hidden characteristics? |
| Few firms and sticky high prices | Oligopoly game | Are there collusion and defection incentives? |

## Common Mistakes

| Mistake | Correction |
|---|---|
| The seller legally pays the tax, so the seller bears it. | Incidence follows elasticity. |
| Trade hurts some workers, so trade reduces welfare. | Total surplus usually rises; distribution needs separate policy. |
| A loss-making firm should always shut down. | Short run depends on whether price covers variable cost. |
| Monopoly profit itself is the social loss. | Deadweight loss comes from blocked mutually beneficial trades. |
| Government provision means public good. | Public good depends on nonexcludability and nonrivalry. |
