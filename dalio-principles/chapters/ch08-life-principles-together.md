# Chapter 8: Life Principles Together

## Core Idea
The life principles become an integrated operating system: see reality clearly, use pain to learn, run the 5-step process, remain radically open-minded, account for human wiring, and decide from explicit principles.

## Frameworks Introduced
- **Integrated personal operating system**:
  - When to use: when multiple principles apply at once.
  - How: combine truth-finding, diagnosis, redesign, and disciplined execution.

## Key Concepts
- **Coherence**: principles reinforce each other.
- **Self-management**: the aim is not inspiration but repeatable judgment.

## Mental Models
- Use the principles as a stack rather than isolated quotes.
- Ask which missing discipline explains the current failure: reality, openness, diagnosis, fit, or execution.

## Anti-patterns
- **Collecting principles without using them**: intellectual agreement is not operational use.
- **Treating one principle as the whole system**: each one covers a different failure mode.

## Worked Example
Dalio's summary section acts like a compressed control panel. It is designed for recall and application, not literary closure.

## Key Takeaways
1. The power is in the system, not the slogan.
2. Principles become useful when they guide live decisions.
3. Consistency compounds.
